# Discord-Selfbot
The only Public Discord Selfbot with Malicious Code. this can destroy a server, spam messages and massively create channels


<b>Installation</b> :<br> 
npm install discord.js<br>
node index.js

<b>config.json</b> :<br> 
Change Token<br>
Change Prefix

<b>index.js</b> :<br> 
Add your Commands.<br>

<b>List of Commands</b> :<br> 
d!kick<br>
d!ban<br>
d!say {message}<br>
d!spam {message}<br>
d!dm {message}<br>
d!destroy<br>
